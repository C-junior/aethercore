## ğŸ›  BALANCE METHODOLOGY

### 1. Data Collection
- Gather relevant metrics (win rates, pick rates, gold flow)
- Identify outliers and trends
- Compare against design targets

### 2. Analysis
- Is this a numbers problem or a design problem?
- Single variable or systemic issue?
- Player skill variance vs. balance issue?

### 3. Adjustment Strategy
- **Buff the Weak** preferred over **Nerf the Strong** (feels better)
- Small incremental changes (10-20%) vs. drastic shifts
- Consider soft counters before hard nerfs

### 4. Validation
- Define expected outcome before implementation
- Plan measurement approach
- Set timeline for evaluation
